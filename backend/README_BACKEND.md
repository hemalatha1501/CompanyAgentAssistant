Company Agent Assistant

Company Agent Assistant is a full-stack web application that simulates a company agent using HTML, CSS, JavaScript, and Node.js backend. It provides an interactive and aesthetic interface for users to chat with a mock AI assistant.

Features

Interactive AI Assistant: Chat with a simulated company agent.

Responsive Frontend: Works seamlessly on desktop and mobile.

Modern UI/UX: Contrasting colors, cute icons, and appealing design.

Lightweight Backend: Handles chat responses and simulates real AI logic.

Future-Ready: Structured for easy integration with real AI APIs.

Technologies Used

Frontend:

HTML, CSS, JavaScript

Icons and color schemes for UI/UX

Backend:

Node.js

Express.js (for routing API endpoints)

Optional: JSON files or in-memory data for mock responses

Installation
1. Clone the repository
git clone https://github.com/your-username/CompanyAgentAssistant.git
cd CompanyAgentAssistant

2. Backend Setup

Navigate to the backend folder (if separated):

cd backend


Install dependencies:

npm install


Start the backend server:

node server.js


By default, it runs on http://localhost:3000 (or the port specified in server.js).

3. Frontend Setup

Open the frontend folder:

cd ../frontend


Open index.html in your browser, or run using Live Server in VS Code.

The frontend communicates with the backend API for simulated AI responses.

Usage

Open the app in your browser (index.html).

Type a message in the chat box.

The backend responds with mock AI replies.

Enjoy interacting with the company assistant!
